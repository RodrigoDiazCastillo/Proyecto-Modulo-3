# Proyecto-Modulo-3
Aplicacion de consola

Proyecto: Calculadora de Gastos en JS

Este es el proyecto final del Módulo 3. El objetivo fue crear una herramienta que funcione en la consola del navegador para ayudar a los usuarios a llevar un control de sus gastos diarios.

¿Qué hace esta aplicación?

La aplicación permite organizar el dinero de forma fácil mediante un menú interactivo:
- Registrar gastos: Puedes ingresar el nombre de lo que compraste y cuánto costó.
- Ver la lista: Muestra en la consola todos los gastos que has guardado hasta el momento.
- Sumar el total: Hace la cuenta matemática de todos tus gastos y te da el resultado final.


¿Cómo lo construí? (Conceptos usados)

Para que el código esté ordenado y cumpla con lo pedido en clase, usé:

1. Variables (let y const): Para guardar los nombres y los montos.
2. Objetos y Arreglos: Cada gasto es un "objeto" con su descripción y precio, y todos se guardan juntos en una "lista" (arreglo).
3. Funciones: Dividí el código en partes pequeñas para que sea más fácil de leer. Hay una función para agregar, otra para mostrar y otra para sumar.
4. Bucles (Loops): Usé un while para que el menú no se cierre solo y un for para revisar la lista de gastos uno por uno.
5. Validaciones: Agregué un pequeño control para que, si alguien escribe letras en lugar de números en el precio, la aplicación le avise que hay un error.


Cómo probarlo

1. Abre el archivo index.html en tu navegador.
2. Abre la Consola de desarrollador (F12 o clic derecho > Inspeccionar).
3. Sigue las instrucciones que aparecen en las ventanitas (prompt).
